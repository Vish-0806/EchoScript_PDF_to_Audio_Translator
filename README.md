# 🎧 EchoScript – PDF to Audio Converter with Translation

EchoScript is a web application that converts PDF documents into natural-sounding audio using AI-powered text-to-speech, with optional multi-language translation support.

---

## 🚀 Features

- 📄 Upload and extract text from PDF files
- 🔊 Convert text into high-quality audio (MP3)
- 🌍 Optional translation using Deep Translator
- 🎤 Multiple voice selection (Edge TTS voices)
- ⚡ Fast parallel audio generation
- 📊 Displays PDF details (pages, word count, estimated duration)
- 🎧 Built-in audio player with download option
- 🎨 Clean and responsive user interface

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** Flask (Python)  
- **PDF Processing:** pdfplumber  
- **Text-to-Speech:** edge-tts  
- **Translation:** deep-translator
